\# PTB-XL Dataset



The PTB-XL dataset is used for ECG-based evaluation in the CAPCAF-XMORL-PUT framework.



Dataset download link:

https://physionet.org/content/ptb-xl/1.0.1/



After downloading, place the files inside this folder:



data/ptbxl/

\# ARAS Smart Home Dataset



This dataset is used for activity recognition and privacy-utility evaluation.



Download from:

https://archive.ics.uci.edu/



After downloading, place files inside:



data/aras/

\# GeoLife Trajectory Dataset



This dataset is used for trajectory-based privacy-utility experiments.



Download from:

https://www.microsoft.com/en-us/research/project/geolife-building-social-networks-using-human-location-history/



Place files inside:



data/geolife/
